Dataset Statistics

This project presents six types of AM dataset, totaling more than 100 questions. They include design, machine-dependent, machine-independent data, monitoring data, post-process, and quality measurement data. These datasets are created from different published papers. Below, we provide some examples of different types of datasets.

Dataset Type Number of Papers Number of Questions Source

Machine Dependent 12 25 Conference Papers
Machine Independent 10 20 Technical Reports
Monitoring Data 8 15 Industry Reports
